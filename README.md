[go to platform](http://platform.xrubik.com/)

## Getting started

```bash
# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9527/platform

## Build

```bash
# build for production environment
npm run build:prod
```

## Advanced

```bash
# code format check
npm run lint

# code format check and auto fix
npm run lint -- --fix
```
